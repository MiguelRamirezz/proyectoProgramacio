2. Datos que Recopilamos

Podemos recopilar y procesar los siguientes datos:

  • Datos de identificación: Nombre, apellidos, DNI/NIF, dirección de correo electrónico, dirección postal, número de teléfono, etc.
  • Datos de pago: Información de la tarjeta de crédito/débito (procesada de forma segura a través de pasarelas de pago).
  • Datos de navegación: Dirección IP, tipo de navegador, páginas visitadas, tiempo de permanencia, etc.
  • Datos de perfil: Preferencias, intereses, historial de compras, etc.

3. Finalidades del Tratamiento

Tus datos personales serán utilizados para las siguientes finalidades:

  • Gestionar tu cuenta de usuario en nuestra plataforma.
  • Procesar y gestionar tus pedidos, incluyendo el envío y la facturación.
  • Atender tus consultas y solicitudes de información.
  • Enviarte comunicaciones comerciales sobre nuestros productos y servicios, si has dado tu consentimiento.
  • Mejorar nuestros servicios y experiencia de usuario.
  • Cumplir con nuestras obligaciones legales.

4. Base Legal para el Tratamiento

El tratamiento de tus datos se basa en:

  • Ejecución de un contrato: Para la gestión de tu cuenta y pedidos.
  • Consentimiento: Para el envío de comunicaciones comerciales.
  • Interés legítimo: Para mejorar nuestros servicios y seguridad.
  • Cumplimiento legal: Para cumplir con nuestras obligaciones legales.

5. Cesión de Datos

Tus datos podrán ser comunicados a:

  • Empresas de transporte para la entrega de pedidos.
  • Proveedores de servicios de pago para el procesamiento de transacciones.
  • Autoridades competentes cuando sea requerido por ley.

6. Derechos de los Usuarios

Tienes derecho a:

  • Acceso: A los datos personales que te conciernen.
  • Rectificación: De datos inexactos o incompletos.
  • Supresión: De tus datos cuando ya no sean necesarios.
  • Limitación del tratamiento: En ciertas circunstancias.
  • Portabilidad: De tus datos a otro responsable.
  • Oposición: Al tratamiento de tus datos en determinados casos.
  • Retirar el consentimiento: En cualquier momento.

Para ejercer estos derechos, envía una solicitud a protecciondedatos@gellverse.com junto con una copia de tu DNI o documento identificativo.

7. Seguridad de los Datos

Implementamos medidas técnicas y organizativas para garantizar la seguridad de tus datos personales y protegerlos contra accesos no autorizados, alteración, divulgación o destrucción.

8. Conservación de Datos

Conservaremos tus datos personales durante el tiempo necesario para cumplir con las finalidades para las que se recopilaron, incluyendo el cumplimiento de obligaciones legales, contables o de informes.

9. Cambios en la Política de Privacidad

Nos reservamos el derecho de modificar esta política de privacidad en cualquier momento. Cualquier cambio será publicado en esta página con la fecha de actualización correspondiente.
